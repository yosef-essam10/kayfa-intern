# Fullstack Development Diploma
### Kayfa Academy — Accredited Professional Program

A hands-on, project-first program that takes you from zero to job-ready in full-stack web
development. You build complete applications, deploy them live, and graduate with a portfolio
that's ready for jobs and freelancing.

| | |
|---|---|
| **Duration** | 6 months |
| **Sessions** | 55 live sessions (2–3 hours each) |
| **Format** | Interactive live sessions — also recorded, available to you for 1 year |
| **Level** | Junior Full Stack Developer |
| **Outcomes** | Live portfolio · Graduation project · CV guidance · A personal Sub Domain per student |

## Prerequisites
- No prior programming experience required — the diploma starts from scratch.
- Basic computer skills, curiosity, and a problem-solving mindset.
- Recommended (optional): basic English for reading technical documentation.
- A laptop that runs VS Code and a browser smoothly (8 GB RAM+), and a stable internet connection.

## What You'll Learn
**Frontend:** HTML5, CSS3, Responsive Design, Bootstrap, Tailwind CSS, Git & GitHub, JavaScript
(DOM/BOM), Async & APIs, TypeScript, React, React Hook Form & Zod, Redux Toolkit, React Query,
shadcn/ui, Next.js.

**Backend:** Node.js, Express.js, REST APIs, MVC, Middleware, MongoDB & Mongoose, MySQL &
Sequelize, Joi Validation, JWT Auth, Roles & Authorization, Sessions & Cookies, Social Login,
File Uploads, Emails, Cron Jobs, PDF & QR, Socket.IO, Redis Caching, Docker, Deployment,
SSL & Sub Domain.

## Curriculum Structure
1. **Frontend Foundations** — Sessions 1–12 — HTML, CSS, Bootstrap, Tailwind, Git/GitHub, first live deploy.
2. **JavaScript & TypeScript** — Sessions 13–22 — logic, DOM, APIs, async, ES6+, OOP, TypeScript.
3. **React Ecosystem** — Sessions 23–31 — components, hooks, forms, routing, Redux Toolkit, React Query, shadcn/ui.
4. **Next.js** — Sessions 32–35 — routing, rendering strategies, API routes, auth concepts, deployment.
5. **Backend (Node.js & Express)** — Sessions 36–50 — REST APIs, databases, auth, advanced features, deployment, graduation project.
6. **Non-Technical Phase** — Sessions 51–55 — English for developers + career development (portfolio, CV, interviews, freelancing).

## Major Projects
- **S07** — Responsive HTML + CSS site (first portfolio piece)
